HTML、CSS 及 JQUERY 函式庫簡介

HTML

超文本標記語言(HyperText Markup Language，簡稱：HTML)是一種用於建立網頁的標準標記語言。 網頁瀏覽器可以讀取HTML檔案，並將其彩現成視覺化網頁。 HTML描述了一個網站的結構語意隨著線索的呈現，使之成為一種標記語言而非程式語言。

CSS

階層式樣式表是一種用來為結構化文件（如HTML文件）添加樣式（字型、間距和顏色等）的電腦語言 其中HTML負責確定網頁中有哪些內容，CSS確定以何種外觀(大小、粗細、顏色、對齊和位置)展現這些元素。 CSS可以用於設定頁面布局、設定頁面元素樣式、設定適用於所有網頁的全域樣式。

JQUERY

jQuery是一套跨瀏覽器的JavaScript函式庫，簡化HTML與JavaScript之間的操作。 jQuery的語法設計使得許多操作變得容易，如操作文件（document）、選擇文件物件模型（DOM）元素、建立動畫效果、處理事件、以及開發Ajax程式。 模組化的方式使jQuery函式庫能夠建立功能強大的動態網頁以及網路應用程式。
